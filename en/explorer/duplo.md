# Duplo Rating

The leaderboard for the **Duplo** game: top players, progress and seasonal results.

## Views

* **Day** — pick any date from the daily archive (history going back to 2024)
* **Season** — seasonal standings
* **All** — all-time ranking

## What the table shows

| Column | Meaning |
| --- | --- |
| **#** | Player rank |
| **Wallet** | Player account (`.tg` / `.near`) |
| **DARAI** | Score in Darai |
| **Boost** | Active boost percentage (top players run 1 000%+) |

## Prize pool

The leaderboard page shows the **prize pool wallet** with a one-tap **Send** button — contribute to the pool directly.
