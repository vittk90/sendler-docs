# Портфолио

Все активы под контролем.

## Вид по коллекциям

Портфолио организовано по коллекциям. Для каждой:

* Вкладки **Owned / Listed** со счётчиками
* Адрес контракта коллекции (копирование в один тап)
* Панель **Orders** и **история** операций
* Поиск, фильтры, сортировка и группировка
* **Переключение коллекций** — перепрыгивай между своими коллекциями, не выходя из портфолио
* **Owners** — тапни на значок рядом с названием коллекции, чтобы увидеть всех держателей NFT этой коллекции, отсортированных по количеству (работает для любой коллекции на платформе, не только твоей)

## Действия с NFT

У каждой карточки два быстрых действия:

* **Send** — отправить на любой аккаунт
* **List** — выставить на продажу

## Массовая отправка — Send NFTs

Флагманский инструмент портфолио:

1. Открой **Send NFTs**.
2. Введи **получателя** (`account.near` или `.tg`).
3. Используй **фильтр по названию** для быстрого поиска.
4. Отправляй по одному — или выбери несколько и отправь одной операцией.

Работает по всему портфолио: в списке каждый твой NFT с иконкой коллекции, названием и token ID.
