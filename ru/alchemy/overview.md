# Alchemy Lab — обзор

> **Transmute NFTs and FTs into new rewards.**

Alchemy Lab — он-чейн крафтинг-движок Sendler. У него две стороны:

* **Крафтеры** выбирают live-рецепт, отдают то, что он просит, и получают награду — всё в одном подтверждении кошелька.
* **Владельцы лабораторий** создают свою лабу, собирают рецепты, пополняют запас наград и следят за аналитикой.

{% hint style="info" %}
**Ничего не сжигается и не теряется.** Если ты заранее внёс предметы под рецепт, их всегда можно забрать из *My items* — даже если рецепт или лаба потом удалены.
{% endhint %}

## Explore

Вкладка **Explore** — пульс Лаборатории:

* Статистика платформы: **Labs**, **Recipes**, **Crafts**, **Crafters**
* **Top Recipes** — самые крафтовые рецепты
* **Top Labs** — самые активные лаборатории
* **Top Crafters** — лидерборд за *всё время* и *эту неделю*

## Labs

Вкладка **Labs** — список всех лабораторий с владельцем, числом рецептов и активностью. Есть поиск, избранное и быстрый шеринг.

## Скидочная сетка Adam

Держишь NFT из коллекции **Adam** — комиссия за крафт снижается автоматически, чем выше редкость, тем больше скидка:

| Редкость | Скидка |
| --- | --- |
| Genesis | −100% |
| Уник | −90% |
| Лега | −50% |
| Эпик | −40% |
| Рар | −30% |
| Анк | −20% |
| Ком | −10% |

{% hint style="info" %}
Genesis — предмет в единственном экземпляре, принадлежит основателю проекта. Все остальные тиры доступны через [Adam Mint](../adam-mint.md).
{% endhint %}

Скидка применяется автоматически в момент крафта — никаких дополнительных шагов.
